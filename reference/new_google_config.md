# Create a new google_config object

Creates a new google_config object

## Usage

``` r
new_google_config(client_id, client_secret, app_url, use_refresh_token = TRUE)
```

## Arguments

- client_id:

  The client ID for the app

- client_secret:

  The client secret for the app

- app_url:

  The URL for the app

- use_refresh_token:

  Enable the use of refresh tokens

## Value

A google_config object
