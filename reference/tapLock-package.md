# tapLock: Seamless Single Sign-on for 'shiny'

Swift and seamless Single Sign-On (SSO) integration. Designed for
effortless compatibility with popular Single Sign-On providers like
Google and Microsoft, it streamlines authentication, enhancing both user
experience and application security. Elevate your 'shiny' applications
for a simplified, unified, and secure authentication process.
